# stm32-blink-uart
User_LED (PC13) mirksi kas vieną sekundę ir priema tik 5 ženklų žinutes per UART 2 (interrupt mode), kurią iškart išspausdina.

Valdiklis: STM32F411 WeAct Blackpill V3.1

USB-TTL: CH340
